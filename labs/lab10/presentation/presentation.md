# Итоги работы

## Вывод

В ходе лабораторной работы были изучены основные команды для управления модулями ядра в операционной системе **Linux**.  
Были выполнены действия по загрузке, выгрузке и просмотру параметров модулей ядра с использованием **modprobe**.  
Исследованы модули **ext4** и **bluetooth**, а также выполнено обновление ядра системы с помощью **dnf**.  
Закреплены навыки администрирования и работы с компонентами ядра.
